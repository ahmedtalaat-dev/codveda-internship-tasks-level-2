# Codveda Internship Tasks - Level 2

This repository contains my **Level 2 tasks** for the Front-End Internship at **Codveda Technologies**.  
It includes **two projects**: a **Photo Gallery** and an **XO Game**.

---

## 1. Photo Gallery

**Description:**  
A simple and responsive photo gallery built using **HTML, CSS, and JavaScript**. Users can view images in a clean, organized layout.

**Features:**  
- Responsive layout using CSS  
- Image hover effects  

**Technologies:**  
HTML | CSS | JavaScript

**Live Demo:** *(https://manara0.netlify.app/)*

---

## 2. XO Game

**Description:**  
A fun and interactive Tic-Tac-Toe (XO) game built with **Next.js, Tailwind CSS, and Lucide React**. Users can play **against a friend** or **against the computer**, with selectable difficulty levels.

**Features:**  
- **Two-player mode:** Play with a friend  
- **Single-player mode:** Play with computer (Easy, Moderate, Hard)  
- Responsive design  
- Modern UI with icons from **Lucide React**

**Technologies:**  
Next.js | Tailwind CSS | Lucide React | JavaScript

**Live Demo:** *(https://xo-game10.netlify.app/)*
